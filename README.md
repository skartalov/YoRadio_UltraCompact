This is custom project for YoRadio using 240x240 display, UDA1334 I2S stereo DAC, Li-Po battery charger, 3 way slide button.

Open the schematic and the board with EaglePCB.

Included are: myoptions.h for this specific board and some nice internet radios in playlist.csv

Use this on your own risk, no any support is included. Please don't ask questions. If you know what you are doing you will be able to build this project by your own. If you have questions - this project is not for you.
